# AI Workshop - OpenEdge ABL Project

This is an OpenEdge ABL project demonstrating the Business Entity pattern and modern application development practices.

## Project Structure

- `src/business/` - Business logic classes and datasets
- `src/` - UI window files
- `dump/` - Database schema files
- `doc/` - Documentation
- `.windsurf/` - Windsurf AI assistant configuration

## Getting Started

This project uses the Sports2000 database schema. See `dump/sports2000.df` for the database definition.

## Technologies

- OpenEdge ABL (Progress)
- Business Entity Pattern
- Modern UI development
